# TajsGraph website

This folder contains a minimal Astro website for the TajsGraph mod.

Develop

1. Install dependencies locally:

   npm ci

2. Run dev server:

   npm run dev

Build

   npm run build

The built files will be written to `dist/`. The repository's GitHub Actions workflow publishes `Mods/TajsGraph/website/dist` to the `gh-pages` branch.
